## About
To do application.
You can add, change, delete task, change task status and change task places.

## Technologies
* react
* redux
* react-dnd
* express
* webpack

## How to start
* git clone https://github.com/nickmat95/ToDo.git
* cd ToDo
* npm install
* node sever.js
* cd ToDo (in another terminal)
* webpack-dev-server --hot (in another terminal)
